**StarCraft: Procedural Universe** é um simulador de exploração espacial épico que combina estratégia, RPG e gerenciamento de colônias. Navegue por uma galáxia vasta e dinâmica, onde cada planeta, missão e encontro é único, impulsionado por inteligência artificial avançada.

---

## 🌌 Visão Geral

Assuma o comando de sua própria nave estelar e molde o destino da galáxia. Comece como um explorador solitário e torne-se o líder de uma rede de colônias prósperas ou um diplomata influente entre facções intergalácticas.

### ✨ Diferencial: IA Generativa (Google Gemini)
Diferente de jogos com conteúdo estático, este projeto utiliza o **Google Gemini SDK** para criar:
- **Lore Dinâmica:** Descrições detalhadas e únicas de planetas e sistemas.
- **Missões Inteligentes:** Objetivos e histórias que se adaptam ao seu progresso.
- **NPCs Reativos:** Diálogos e interações gerados em tempo real com personalidades distintas.
- **Eventos de Colônia:** Dilemas éticos e desafios de gerenciamento orgânicos.

---

## 🛠️ Funcionalidades Principais

| Categoria | Descrição |
| :--- | :--- |
| **Exploração** | Mapa da galáxia procedural, sistemas estelares variados e exploração de anomalias espaciais. |
| **Gerenciamento** | Funde e governe colônias, construa infraestrutura e defina políticas governamentais. |
| **Combate** | Sistema de combate tático espacial com gerenciamento de escudo e integridade de casco. |
| **Customização** | Construa e modifique naves no hangar com diversos módulos e cascos especializados. |
| **Diplomacia** | Interaja com facções, assine tratados e gerencie sua reputação intergaláctica. |
| **Progressão** | Árvore de habilidades (Skills) e Árvore Tecnológica (Tech Tree) para desbloquear novas possibilidades. |
| **Economia** | Mercado dinâmico para compra e venda de recursos, materiais de crafting e itens raros. |

## 💻 Tecnologias Utilizadas

O projeto utiliza uma stack moderna para garantir performance e escalabilidade:

- **Frontend:** React 19 (Hooks, Context API)
- **Linguagem:** TypeScript (Tipagem forte para segurança de dados)
- **Estilização:** Tailwind CSS (Design responsivo e temática Sci-Fi)
- **Gráficos:** Three.js (Visualização 3D de elementos espaciais)
- **Build Tool:** Vite (Inicialização rápida e HMR)
- **IA:** Google Generative AI (@google/genai)
- **Ícones:** Lucide React

## 🚀 Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/starcraft-procedural.git
   cd starcraft-procedural
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Configure as chaves de API:**
   Crie um arquivo `.env` na raiz e adicione sua chave do Gemini:
   ```env
   GEMINI_API_KEY=sua_chave_aqui
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Acesse no navegador:**
   `http://localhost:3000`
